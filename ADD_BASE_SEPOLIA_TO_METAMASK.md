# How to Add Base Sepolia Testnet to MetaMask

## Step-by-Step Instructions

### Method 1: Manual Addition (Recommended)

#### Step 1: Open MetaMask
1. Click the **MetaMask extension** icon in your browser
2. Unlock MetaMask with your password if needed

#### Step 2: Open Network Settings
1. Click the **network dropdown** at the top (it probably says "Ethereum Mainnet" or another network)
2. Scroll down and click **"Add Network"** or **"Add a network manually"**

**Alternative way:**
- Click the **three horizontal lines (â˜°)** menu icon (top left)
- Click **"Settings"**
- Click **"Networks"** in the left sidebar
- Click **"Add Network"** button
- Click **"Add a network manually"**

#### Step 3: Enter Network Details
Fill in the following information:

**Network Name:**
```
Base Sepolia
```

**New RPC URL:**
```
https://sepolia.base.org
```

**Chain ID:**
```
84532
```

**Currency Symbol:**
```
ETH
```

**Block Explorer URL (Optional):**
```
https://sepolia-explorer.base.org
```

#### Step 4: Save
1. Click **"Save"** button
2. MetaMask will automatically switch to Base Sepolia network
3. You should see "Base Sepolia" in the network dropdown

### Method 2: Quick Add (If Available)

Some MetaMask versions have a "Quick Add" option:

1. Click network dropdown
2. Click **"Add Network"**
3. Look for **"Base Sepolia"** in the list
4. If you see it, click **"Add"**
5. If not, use Method 1 above
